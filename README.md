## Installation
1. Clone the repository
   ```bash
   git clone https://github.com/ramirezj5/portfolio.git

2. Install dependencies
   ```bash
   cd portfolio
   npm i

2. Define environment variables in .env file
   - MAILGUN_DOMAIN
   - MAILGUN_API_KEY

2. Run development client server and backend server
   ```bash
   npm run start
